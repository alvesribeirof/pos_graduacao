# Exemplo 06 - Prompt Engineering para Scaffolding e Testes

Exemplo focado em prompts para automacao de setup de testes E2E com Playwright.

## Conteudo

- `example.mcp.json`: exemplo de configuracao MCP.
- `prompts/project-scaffolding.md`: roteiro para scaffold de projeto com Playwright e CI.
- `prompts/generate-tests.md`: prompt para geracao de testes de formulario.

## Objetivo

- Definir prompts reutilizaveis para gerar estrutura de testes.
- Padronizar configuracao de CI para rodar somente Chromium.
- Servir como base para exercicios de automacao orientada por prompt.

## Como usar

1. Abra os prompts na pasta `prompts/`.
2. Execute os comandos sugeridos no projeto alvo.
3. Ajuste os testes gerados para o contexto da aplicacao.
