# Exemplo 07 - Browser Automation Prompt

Exemplo com prompt de automacao de navegador para coleta e preenchimento assistido.

## Conteudo

- `example.mcp.json`: configuracao de exemplo para uso com MCP.
- `prompt.md`: prompt com fluxo de navegacao e preenchimento sem envio final.

## Objetivo

- Demonstrar automacao guiada por prompt em paginas reais.
- Praticar extracao de dados e preenchimento de formulario com validacao manual.

## Como usar

1. Abra `prompt.md`.
2. Execute o fluxo em um agente com suporte de browser automation.
3. Revise os campos preenchidos antes de qualquer envio manual.
